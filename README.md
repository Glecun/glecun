# glecun (Nuxt SSG)

Personal website built with Nuxt and static generation.

## Prerequisites

- Node.js 24+
- npm

## Installation

```bash
npm install
```

## Local launch

```bash
npm run dev
```

The site is served with `baseURL: /v2/`, so the local URL is `http://localhost:3000/v2`.

## Build static

```bash
npm run generate
```

The result is in `.output/public/`.
